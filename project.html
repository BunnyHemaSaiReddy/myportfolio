Image Detection and Facial Recognition

This project leverages computer vision techniques using facial recognition methods such as eigenfaces, PCA, and machine learning classifiers (e.g. Decision Trees and MLP). The system processes images by converting them to grayscale, resizing to a fixed dimension, flattening into feature vectors, and then reducing dimensions to highlight key features before classification.

Dataset Details: Number of samples: 190. Total dataset size: n_samples = 190, n_features = 90,000, n_classes = 190

Processing Details: Extracted the top 15 eigenfaces from 142 faces. Projected the input data on the eigenfaces orthonormal basis. Project completed successfully.

Model Evaluation: Accuracy: 95.83%

This system integrates a full pipeline from image processing to model training, offering robust performance for facial recognition tasks. It can be extended to different domains requiring object detection or identification.
